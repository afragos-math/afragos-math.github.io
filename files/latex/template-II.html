Usage

Un-zip the rar folder (you must have an appropriate software to be able to do so). You write on main.tex, which is the main tex file.

structure.tex is reserved for the structure of the document and in the notation.tex all the various new commands can be stored.

Any picture used in the document must be put into the "pictures" folder.
