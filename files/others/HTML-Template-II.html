Usage

Un-zip the rar folder (you must have an appropriate software to be able to do so). All the contents must be put (if not already) in a folder. The template is supposed to be a super simple HTML-CSS template, free of any javascript, for the everyday user.

index.html is the main file, while contact.html and upcoming.html are the subpages. styling.css is the structure file.

Any picture used in the document must be put into the "images" folder. The font (GFS Didot) is located in the "fonts" folder.
